# Back-end
Back-end for a course rating platform.

`npm install` <br/>
`npm run dev` <br/>

The application uses `dotenv` and requires a file named `.env` with secret environmental variables. Include `MONGO_DB_LINK` and `SESSION_KEY` to run the application.
